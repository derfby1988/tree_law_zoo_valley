## วิธีใช้งาน

### เปิดใช้งานสูตร:
1. ไปที่หน้า **จัดการคูปอง & โปรโมชั่น**
2. เลือกแท็บ **"ตั้งค่าสูตรแนะนำ"**
3. เลือกสูตรที่ต้องการจาก Dropdown
4. กด **"เปิดใช้งาน"**

### สร้างสูตรใหม่:
1. กด **"สร้างสูตรใหม่"**
2. ตั้งชื่อและรายละเอียด
3. ปรับน้ำหนักปัจจัย (รวมต้อง = 100%)
4. ตั้งค่าช่วงเวลาใช้งาน
5. บันทึก

### สูตรที่ Active จะถูกใช้:
- ในการคำนวณคะแนนแนะนำสินค้า
- ในการจัดลำดับสินค้าแนะนำ
- ในการแนะนำ % ส่วนลด

## หมายเหตุ
- ใช้ได้ทีละ 1 สูตรเท่านั้น
- สูตรที่มีช่วงเวลาจะหมดอายุอัตโนมัติ
- สูตรที่ active จะ fallback ไปสูตร default ถ้าไม่มี active
- ทุกการเปลี่ยนแปลงจะถูกบันทึกประวัติ
